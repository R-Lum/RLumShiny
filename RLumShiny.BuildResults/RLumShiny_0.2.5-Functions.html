1 app_RLum Run Luminescence shiny apps A wrapper for shiny::runApp to start interactive shiny apps for the R package Luminescence. The RLumShiny package provides a single function from which all shiny apps can be started: app_RLum() . It essentially only takes one argument, which is a unique keyword specifying which application to start. See the table below for a list of available shiny apps and which keywords to use.
